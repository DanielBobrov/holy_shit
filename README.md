# Iterative Reasoning Model

Модель трансформера с итеративным подходом к рассуждению, основанная на концепциях, описанных в работе "Grokked Transformers are Implicit Reasoners".

## Структура проекта

- `model.py`: Основная модель с архитектурой итеративного рассуждения
- `train_model.py`: Скрипт для обучения модели
- `evaluate_model.py`: Скрипт для оценки модели на тестовых данных
- `viz_tools.py`: Инструменты для визуализации внимания и процесса рассуждения
- `run_experiment.py`: Скрипт для запуска полного эксперимента
- `kaggle_run.ipynb`: Jupyter notebook для запуска в Kaggle
